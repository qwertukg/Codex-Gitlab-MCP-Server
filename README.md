# Codex Gitlab MCP Сервер

Проект предоставляет сервер MCP, который соединяет Codex с GitLab. 
Когда задача в GitLab получает метку `todo-dev`, сервер автоматически создаёт merge request с реализацией описанной задачи. 
Все дальнейшие комментарии обрабатываются через заметки MR.

## Запуск

```bash
pip install -e .
codex-gitlab-server
```

## Переменные окружения

- `GITLAB_URL` — адрес инстанса GitLab.
- `GITLAB_TOKEN` — персональный токен доступа.
- `GITLAB_PROJECT_ID` — идентификатор проекта.

Сервер использует Python и объектно-ориентированный подход.
